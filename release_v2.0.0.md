# Release v2.0.0

本版本是 InsightGraph 从「可用」迈向「可长期使用的第二大脑」的重要里程碑：不仅补齐了 AI 写作/检索/溯源链路的核心体验，还引入了块级高亮引用与一套商业级多主题外观系统。

## ✨ 亮点速览
- **AI Writer（AI 魔法棒）**：模板模式 + 完全自定义模式，内置 11 种写作模板，SSE 流式生成。
- **AI Librarian（AI 图书管理员）**：写作时防抖检索历史知识库，侧边栏主动推荐相关胶囊/文献。
- **Block-level Reference（块级高亮引用）**：阅读时划选提取高亮胶囊；写作时输入 `((` 弹出搜索插入引用块，显著降低 Token 并减少幻觉。
- **Dynamic Themes（动态多主题）**：全站暗黑模式彻底生效，新增 Forest / Latte / Geek / Apple Glass（毛玻璃）等主题。
- **多租户与权限隔离**：引入 User、JWT 登录、owner_id + visibility 数据隔离基础能力。

## 🧠 AI 与写作体验
- **AI 魔法棒双模式**
  - 模板模式：润色 / 改写 / 扩写 / 压缩 / 大纲 / 要点 / 行动清单 / Q&A / 记忆卡片 / 博客 / 卡片。
  - 自定义模式：完全由用户 Prompt 驱动。
- **流式生成**：前端打字机效果，后端通过 SSE 转发 Dify 流式输出。
- **AI Librarian 推荐**：在写作过程中后台检索（防抖），把最相关的历史胶囊/文献片段推送到侧边栏。

## 🎯 块级引用（降低幻觉的关键）
- **阅读侧提取**：在文献阅读/翻译区域划选文本，一键保存为带溯源信息的「高亮胶囊」。
- **写作侧引用**：在每日笔记中输入 `((` 呼出搜索窗，选择后以 Markdown 引用块形式插入。

## 🎨 外观与主题
- **暗黑模式全站生效**：从“仅少数组件变暗”升级为全局语义化主题（CSS Variables + Tailwind 映射）。
- **五大主题**：
  - Default（科技紫）
  - Forest（护眼森林）
  - Latte（复古拿铁）
  - Geek（极客纯黑）
  - Apple Glass（果味毛玻璃：`backdrop-filter` + 半透明材质）

## 🔐 基础安全与工程化
- **JWT 登录体系**：路由权限、管理员管理、用户必改密码策略等基础安全能力。
- **数据隔离字段**：核心表引入 owner_id + visibility（public/private），为团队模式奠定基础。

## 📦 安装与使用
### 桌面端
- macOS：下载 Release 里的 `InsightGraph-*.zip`，解压后拖入「应用程序」即可。
- Windows：下载 Release 里的 `InsightGraph Setup *.exe`，双击安装。

### 后端服务（必须）
当前桌面端默认连接本机 `http://localhost:8000`，请先在本地启动后端：
- 使用 `docker-compose.example.yml` + `.env.example` 完成配置后运行 Docker Compose。

## ✅ 校验
- 你可以在「系统设置 -> 外观与基础」里切换暗黑模式与 5 套主题。
- 在「每日笔记」中输入 `((` 检查块级引用弹窗。
- 在「今日资讯」精读页划选一段文本，检查“提取为高亮胶囊”。

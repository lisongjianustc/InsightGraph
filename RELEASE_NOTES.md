# 🎉 InsightGraph v1.0.0 - 个人第二大脑的基石之作

我们非常激动地宣布 **InsightGraph v1.0.0** 首个正式版本发布！

InsightGraph 旨在解决个人知识管理中的核心痛点：**信息孤岛**与**检索困难**。在这个版本中，我们成功构建了一个以大模型 (LLM) 为核心、全自动化的多模态知识处理引擎。它能够帮你捕获从长篇学术论文到一闪而过的截图碎片，并通过动态知识图谱和全局问答将它们彻底贯通。

## 🌟 v1.0.0 核心特性

### 1. 📡 自动知识发现 (Auto-Feed)
* **定时抓取**：内置 `APScheduler`，后台每小时自动拉取 arXiv 等 RSS 订阅源的前沿文献。
* **一键泛读与精读**：调用大模型实现极速摘要提取（Skim Reading）和全文级别的深度双语精读解析（Deep Reading），自动落库并向量化。

### 2. ⚡️ 多模态闪念胶囊 (Multi-Modal Capsules)
* **全格式文档解析**：支持 PDF、Word、Excel、PPT、TXT、MD 文件的直接拖拽入库。
* **图片 OCR 智能提取**：支持剪贴板截图直接粘贴 (`Cmd+V`)，系统调用多模态大模型（如 Qwen-VL、Moondream）自动提取图片中的文本、表格及公式并转为精美的 Markdown。

### 3. 🌌 动态概念图谱 (Knowledge Graph)
* **AI 自动实体抽取**：后台静默调用大模型，从文献和胶囊中自动提取核心概念（Tag）并建立图谱关联。
* **力引导网络可视化**：基于 ECharts 实现流畅的节点聚类，只显示核心概念，大幅降低图谱视觉噪音。
* **动态释义下钻**：点击概念节点，大模型将为你**动态生成该专业名词的严谨定义**。并可通过抽屉一键跳转至关联的原文献或胶囊碎片。

### 4. 🤖 跨源全局问答 (Global Chat)
* **打通全库数据**：一个入口即可对话你在 InsightGraph 中保存的所有原文库、精读库和闪念胶囊。
* **多模态流式提问**：支持上传图片/文档向知识库发起提问。采用极速 SSE 流式输出，且在回答末尾提供**严格的超链接溯源（Citations）**，彻底告别模型幻觉。

### 5. ⚙️ 开箱即用 (Ready to Deploy)
* 提供了一键安装脚本 `./setup.sh`，无缝拉起 FastAPI、PgVector 数据库和 Vue 3 前端。
* 提供内置系统设置页，支持夜间模式（Dark Mode）、手动同步控制与可视化的标签管理。

## 🚀 如何安装？
1. 确保安装了 Docker 和 Node.js
2. 克隆本仓库并配置 `.env` 文件（填入 Dify 对应的 API Key）
3. 运行 `./setup.sh`
4. 访问 `http://localhost:5173` 开启你的知识连接之旅！

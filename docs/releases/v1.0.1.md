# 🚀 InsightGraph 发版说明 - v1.0.1

**发布日期:** 2026年4月

我们非常高兴地宣布 **InsightGraph v1.0.1** 正式发布！
本次更新在阅读体验、用户交互以及跨平台访问能力上实现了巨大飞跃。我们将 InsightGraph 从一个强大的 Web AI 知识工具，彻底进化为一款拥有所见即所得编辑能力的**桌面级高质感应用程序**。

---

## 🌟 核心更新亮点

### 1. 🖥️ 全新桌面端体验 (Electron App)
InsightGraph 不再局限于浏览器标签页中。
- **独立桌面应用**：我们将整个前端成功打包为基于 Electron 的高性能 macOS 原生应用。
- **无干扰的沉浸感**：独立的程序坞(Dock)图标、隐藏式沉浸标题栏以及原生窗口管理，为你带来真正的“第二大脑”沉浸式体验。
- **专属 "iG" 脉冲图标**：全新设计了高质感的 Squircle（圆角矩形）应用图标，融合了科技脉冲与 `iG` 专属标识。
- **原生 PDF 渲染引擎**：在桌面端点击 PDF 链接不再跳转外部浏览器，而是直接唤起一个带有原生 PDF 阅读器插件的独立 Electron 子窗口，阅读流丝滑不中断。

### 2. 📝 "Typora 级" Markdown 沉浸编辑
我们彻底重构了“闪念胶囊”的编辑与阅读体验。
- **ByteMD 强力驱动**：废弃了简陋的文本框，引入字节跳动开源的 ByteMD，带来真正的 **所见即所得 (WYSIWYG)** 极客编辑体验。
- **实时渲染引擎**：完美支持 GitHub 风格语法 (GFM)、代码块高级高亮，以及 LaTeX 数学公式实时渲染。
- **沉浸式阅读模式**：为阅读面板定制了类似高级出版物的排版主题，字号、行距全面优化，让长篇笔记的阅读成为一种享受。

### 3. 📄 智能文档阅读器无缝切换
回顾历史文献与附件的体验前所未有地优雅。
- **分段式切换器**：打开包含原文件（PDF、Word、Excel、PPT）的胶囊时，顶部将出现一个精美的圆角分段切换按钮。
- **原文件 vs 解析文本**：只需一键，即可在 **[原文件原生排版预览]** 与 **[AI 提取解析的纯文本 Markdown]** 之间极速无缝横跳。

### 4. 🕸️ 知识图谱深度导航
在思维网络中的探索变得更具行动力。
- **操作导向的节点卡片**：在图谱中点击任意节点，侧边栏关联的文档卡片底部新增了专属操作栏。
- **智能路由跳转**：
  - 对于 **文献 (Original)**：可选择“查看来源网页(如 arXiv)”或“直接在本地拉起原 PDF”。
  - 对于 **胶囊 (Capsules)**：可选择“跳转至胶囊详情”或“直接打开胶囊关联的本地原文件”。

### 5. 🐛 Bug 修复与稳定性提升
- 修复了对话模块重构时残留的 `GlobalMessage` 导入导致的后端启动崩溃死循环问题。
- 修复了 ByteMD 编辑器在 Vue 3 下 `v-model` 绑定失效导致编辑模式内容为空的问题。
- 解决了 Electron 打包后白屏的问题（将 Vite 基础路径改为相对路径，并将 Vue Router 切换为 `Hash` 模式）。
- 修复了桌面端因 API 路径相对化导致的请求失败问题（硬编码绝对路径以穿透 Docker 容器进行通信）。

### 6. 🛠️ 全新 DevOps 运维脚本集
我们为项目增加了极具专业度的灾备与自动化部署能力。
- **自动备份 (`scripts/backup.sh`)**：支持一键完整备份 PG 数据库（含高维向量）、本地附件、n8n 工作流数据及环境变量，并支持自动清理旧备份。
- **灾难恢复 (`scripts/restore.sh`)**：支持从指定备份文件夹一键安全回滚，自动切断连接、重置库表并恢复所有文件。
- **一键初始化 (`scripts/init.sh`)**：在全新机器上只需一条命令，即可全自动检查环境、创建本地挂载目录、赋予特殊权限并拉起容器群。

## 🔧 升级指南
如果您正在通过 Docker 本地运行项目：
1. 从 `main` 分支拉取最新代码。
2. 重启后端服务：`docker compose restart backend`。
3. 若要体验全新的桌面端应用，请进入 `frontend` 目录并执行：
   ```bash
   npm run electron:build
   ```
   编译完成后，您的全新 macOS 应用将生成在 `frontend/dist_electron/mac-arm64/` 目录中。双击 `InsightGraph.app` 即可开启全新的桌面端体验。

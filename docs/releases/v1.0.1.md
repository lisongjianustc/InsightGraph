# 🚀 InsightGraph Release Notes - v1.0.1

**Release Date:** April 2026

We are thrilled to announce the release of **InsightGraph v1.0.1**! 
This update brings a massive leap in reading experience, user interaction, and cross-platform accessibility. We've transformed InsightGraph from a powerful web-based AI knowledge tool into a polished, immersive **Desktop-Class Application** with WYSIWYG editing capabilities.

---

## 🌟 What's New in v1.0.1

### 1. 🖥️ The New Desktop Experience (Electron App)
InsightGraph is no longer confined to your browser tabs. 
- **Standalone Desktop App**: We have successfully wrapped the entire frontend into a high-performance Electron application for macOS.
- **Distraction-Free Immersion**: Enjoy a true "Second Brain" experience with a dedicated dock icon, hidden title bars, and native window management.
- **Custom "iG" Pulse Icon**: We designed and integrated a brand new, highly sophisticated, squircle-shaped app icon featuring a glowing `iG` pulse monogram.
- **Native PDF Rendering**: Clicking on a PDF link inside the desktop app no longer kicks you out to Safari; it now spawns a dedicated, native Electron sub-window equipped with a built-in PDF viewer for a seamless reading flow.

### 2. 📝 "Typora-Level" Markdown Experience
We completely overhauled the Capsule (Quick Note) editing and reading experience.
- **ByteMD Integration**: Replaced the basic textarea with ByteMD (by ByteDance), bringing a true **WYSIWYG (What You See Is What You Get)** editing experience.
- **Live Rendering**: Support for GitHub Flavored Markdown (GFM), syntax highlighting for code blocks, and real-time LaTeX math formula rendering.
- **Immersive Reading Mode**: Upgraded the viewing panel with a clean, beautifully spaced typography theme inspired by premium publishing platforms, offering an unparalleled reading experience for your long-form notes.

### 3. 📄 Smart Document Viewer Toggle
Reviewing past literature and attachments has never been this elegant.
- **Segmented Control Switch**: When opening a capsule that contains an original file (PDF, Word, Excel, PPT), a sleek new toggle appears at the top.
- **Original vs. Parsed**: You can now seamlessly switch between viewing the **[Original Document Layout]** (rendered via iframe or vue-office) and the **[AI-Parsed Markdown Text]** with just one click.

### 4. 🕸️ Enhanced Knowledge Graph Navigation
Exploring your network of thoughts is now more actionable.
- **Action-Oriented Nodes**: When clicking on a node in the Graph, the associated document cards in the sidebar now feature specific action buttons.
- **Smart Routing**: 
  - For **Original** literature: Choose between opening the external source URL (e.g., arXiv) or directly launching the local PDF.
  - For **Capsules**: Choose between jumping to the capsule's detail view or opening its original attached file.

### 5. 🐛 Bug Fixes & Stability
- Fixed a critical backend crash loop caused by a phantom import of `GlobalMessage` during the chat module refactoring.
- Fixed an issue where the ByteMD editor would show empty content in edit mode due to incorrect Vue 3 `v-model` binding.
- Resolved the blank screen issue when running the packaged Electron app by switching Vite's base path to relative and migrating Vue Router to `createWebHashHistory`.
- Corrected API base URL routing in the Electron environment to ensure the Feed component correctly fetches data from the backend Docker container.

---

## 🔧 Upgrading
If you are running the project locally via Docker:
1. Pull the latest code from the `main` branch.
2. Restart your backend services: `docker compose restart backend`.
3. To experience the new desktop app, navigate to the `frontend` folder and run:
   ```bash
   npm run electron:build
   ```
   Your new macOS application will be generated in the `frontend/dist_electron/mac-arm64/` directory.
